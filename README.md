# recipe-management-website
A Django-based Recipe Management Website that allows users to register, log in, and manage recipes efficiently. Implemented CRUD operations, authentication system, middleware integration, and dynamic templates for seamless user interaction. Built using Django, Python, SQLite, HTML/CSS, and Bootstrap.
